# Hweb template
## Editing
| Keyword | Defenition |
| :-- | :-- |
| ` {{YOURNAME}} `| Your name |
| ` {{TOPIC 1-4}} ` | The topic of the current section |
| ` {{PARAGRAPH 1-4}} ` | The content of the current section |

Use the config.json for editing NOT DIRECLY IN THE HTML, otherwise your data will be overwritten.
